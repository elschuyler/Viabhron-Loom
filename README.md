# Viabhron: The Shadow Loom

PWA live at https://elschuyler.github.io/Viabhron-Loom/

## 🌌 Vision
Viabhron is a multi-perspective, celestial-organic environment where code is not manufactured, but **cultivated**. The world is a tapestry of nature, stars, and electrical grids woven together by the **Loom Scribe**.

## 🛠️ The Loom (PWA)
The root of this repository contains a mobile-optimized PWA designed for "knitting" game data on a low-RAM Android phone.
- **World Root**: Forge location and connection seeds.
- **Life Seed**: Cultivate creature and entity data.
- **Soul Weaver**: Weave character identities and drive codes.
- **World Pulse**: Monitor the geopolitical and trade-vine currents of the world.

## 📁 Library Structure
- `/library/roots/`: Location data and grid geometry.
- `/library/seeds/`: Bestiary and creature data.
- `/library/souls/`: Character identities and trauma codes.
- `/library/pulse/`: Geopolitical and business logic.

## 🧠 Prompts
The `/prompts/` folder contains the "Scribe Brain" instructions to help AI extract data from novels and wikis directly into the Loom's JSON format.
